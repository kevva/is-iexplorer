# is-iexplorer

> Check if browser is Internet Explorer

Supports IE6, IE7, IE8, IE9, IE10 and IE11.


## Install

```
$ npm install --save is-iexplorer
```


## Usage

```js
var isIexplorer = require('is-iexplorer');

console.log(isIexplorer);
//=> false (hopefully)
```
